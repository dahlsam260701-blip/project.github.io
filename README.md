# Landing Page — Магазин брендовых кроссовок

Простой лендинг с:
- Заголовком
- Подзаголовком
- Картинкой справа
- Кнопкой "Купить"

## Запуск

Откройте файл index.html в браузере.

## Структура

- index.html — разметка
- styles.css — стили
- img/sneakers.png — изображение
